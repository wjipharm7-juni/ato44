# 황박사의 지혜로운 건강교실 - 치료 사례 랜딩페이지

## 프로젝트 개요
황박사의 지혜로운 건강교실 유튜브 광고용 랜딩페이지입니다. 아토피 치료 전후 비교 사진을 통해 치료 효과를 시각적으로 보여주는 웹페이지입니다.

## 주요 기능

### ✅ 완료된 기능
1. **성인 치료 사례**
   - 얼굴 치료 전후 비교
   - 상체(가슴/목) 치료 전후 비교
   - 하체(다리) 치료 전후 비교
   - 무릎 치료 전후 비교

2. **어린이 치료 사례**
   - 팔 치료 전후 비교
   - 무릎 뒤 치료 전후 비교
   - 팔꿈치 치료 전후 비교
   - 완치 감사 문자 표시

3. **치료 후 근황**
   - 북경에서 치료받은 분의 건강한 근황
   - 감사 이메일 4건 표시

4. **디자인 특징**
   - 현대적이고 깔끔한 그라데이션 디자인
   - 치료 전/후 명확한 시각적 구분
   - 화살표로 변화 과정 직관적 표시
   - 반응형 디자인 (모바일 최적화)
   - 친근하고 따뜻한 색상 사용

5. **상담 정보**
   - 상담 시간: 10:00 - 18:00
   - 상담 전화: 031-439-6557
   - 헤더와 푸터에 연락처 정보 표시

- CSS3 (그라데이션, Flexbox, Grid)
- Google Fonts (Noto Sans KR)
- 반응형 웹 디자인

## 디자인 특징

### 색상 팔레트
- **헤더/푸터**: 보라색 그라데이션 (#667eea → #764ba2)
- **치료 전 라벨**: 분홍-레드 그라데이션 (#f093fb → #f5576c)
- **치료 후 라벨**: 블루-시안 그라데이션 (#4facfe → #00f2fe)
- **감사 문자 박스**: 오렌지-피치 그라데이션 (#ffecd2 → #fcb69f)
- **후속 섹션**: 라이트 블루 그라데이션 (#e0f7fa → #b2ebf2)
